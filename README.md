## 🎯 このサイトについて

Llamune Codeは、完全クローズド環境で動作するローカルLLMコーディング支援プラットフォームです。

このコンセプトページでは、**実際のLLMを動かさず**に、以下の特徴を紙芝居的に体験できます：

- 🔒 完全クローズド環境の安心感
- 📁 プロジェクトディレクトリ統合（Function Calling）
- 🧠 推論モデルの思考過程表示
- 🔐 エンドツーエンド暗号化

---

## 🚀 ローカル開発

### 前提条件

- Node.js 18.x 以上

### セットアップ

```bash
# リポジトリをクローン
git clone https://github.com/YOUR_USERNAME/llamune-concept.git
cd llamune-concept

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。
